# ubuno

(Ubu)ntu based Docker image with (No)de.js.

## Info

* Based on Ubuntu LTS (22.04)
* Node.js (16)
* [confd](https://github.com/kelseyhightower/confd) onboard
* other tools:

  * curl
  * wget
  * git
  * gcc (helpful for some node packages - eg. node-gyp)
  * tree
